what is html?

The hypertext markup language or HTML is the standard markup language

for documents designed to be displayedin web browser. it defines the

meaning and structure of web content. It is often assisted by tchnologies

such as cascading style sheets (css) and scripting languages such as Javascript.

Web browsers receive HTML documnts from a web server or from local storage and

render the documents into multimediaweb pages. HTML describes the structure of a

web page semantically and originally included cues its appearance.
